# Verify Bot

Telegram bot যা ছবিতে Verified ব্যাজ লাগিয়ে দেয় এবং হ্যান্ড জেসচার সাপোর্ট করে।

## Deploy করার উপায় (Render / Railway):

1. এই রিপোজিটরি GitHub-এ আপলোড করো
2. Render.com বা Railway.app-এ নতুন Web Service বানাও
3. Environment Variables যোগ করো:
   - Key: `BOT_TOKEN`
   - Value: তোমার Telegram BotFather থেকে পাওয়া TOKEN
4. Deploy করো

সমস্যা হলে বলো!
